# GlowLuxe Lotion E-Commerce Web App

This project is a complete e-commerce web application for a lotion-selling company. It includes:

- Responsive product catalog
- Product details page
- Shopping cart with quantity updates
- Checkout and order confirmation
- SQLite-based order storage
- Docker support and CI workflow

## Run locally

1. Install dependencies: `npm install`
2. Start the app: `npm start`
3. Open: http://localhost:3000

## Test

- Run: `npm test`

## Docker

- Build: `docker build -t glowluxe-app .`
- Run: `docker run -p 3000:3000 glowluxe-app`
